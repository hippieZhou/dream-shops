
## Database setup

```bash
docker run -itd --name mysql-test -p 3306:3306 -e MYSQL_ROOT_PASSWORD=123456 mysql
```

## Run

```bash
./mvnw spring-boot:run
#visit http://localhost:8080/swagger-ui/index.html
```
